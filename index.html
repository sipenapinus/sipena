Simulasi Input Mandor

Bagian ini menyimulasikan tampilan aplikasi smartphone **SIPENA PINUS** yang dipegang oleh mandor di lapangan.

Alur Kerja Form Lapangan:

  • Mandor berkeliling petak untuk memeriksa penyadap.
  • Mandor menginput tanggal, nama penyadap, kode petak, dan estimasi hasil timbangan (kg).
  • Jika kondisi tidak normal (Hujan, Wadah Rusak, dll.), mandor mencantumkan kendala.
  • Data dikirim secara otomatis ke Google Sheets Cloud database Anda.
  • Mode Offline: Bila mandor berada di dalam hutan pinus tanpa sinyal internet, data input tersimpan lokal di HP. Begitu HP mendeteksi sinyal internet kembali, data otomatis sinkron terunggah!
